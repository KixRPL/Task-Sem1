    Tentang HTML

    HTML adalah bahasa markup untuk menyebarkan informasi pada Web. HTML singkatan dari (HYPERTEXT MARKUP LANGUAGE) merupakan Bahasa yang digunakan untuk menampilkan informasi pd halaman web. bahasa ini menggunakan tanda (markup) untuk menandai perintah-perintahnya menentukan tampilan suatu teks dan tingkat kepentingan dari teks tersebut dalam suatu dokumen.

    HTML diajarkan dalam berbagai tingkat pendidikan, mulai dari sekolah dasar hingga perguruan tinggi. Di sekolah dasar, HTML biasanya diajarkan sebagai pengenalan dasar pemrograman web. Di tingkat yang lebih tinggi, HTML diajarkan sebagai bagian dari mata kuliah pemrograman web atau desain web.
