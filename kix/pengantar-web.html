    Kata Pengantar WEB

    Selamat datang di mata kuliah RPL105 Pemrograman Berbasis Web. Mata kuliah ini membahas konsep dasar pengembangan web, meliputi penggunaan HTML untuk struktur dan formulir web, CSS untuk tampilan dan layout, dan JavaScript untuk pengontrolan dan manipulasi elemen web.

    Matakuliah ini juga mencakup fundamental dan implementasi PHP dalam pengembangan web, pengelolaan form, session, dan cookie menggunakan PHP. Selain itu, mahasiswa akan mempelajari implementasi operasi CRUD dengan PHP dan MySQL, serta topik autentikasi dan otorisasi dalam pengembangan aplikasi web. Mata kuliah ini akan dilaksanakan secara daring dan Luring.
